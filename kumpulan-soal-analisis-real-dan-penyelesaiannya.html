13. bagaimana pembuktian analisis real a.0=0


Anggap kita memiliki bilangan a dan b

Anggap teorema sudah dibuktikan bahwa untuk b bilangan real, maka berlaku :
0 + b = b

Sehingga,

a*(0 + b) = a*b
a(0) + a(b) = a(b)

Manipulasi nilai,

a(0) + a(b) - a(b) = a(b) - a(b)
a(0) = 0

Terbukti bahwa a(0) = 0
a*0 = 0

19. tentukan himpunan selesaian dari pertidak samaan berikut dengan x adalah anggota himounan bilangan real . kemudian lukiskan penyelesaiannya dalam garis bilangan , soal -8 ≤ 2/5 ( k - 2 )


Diberikan pertidaksamaan -8 ≤ 2/5 (k – 2), dengan k ∈ bilangan real.
Penyelesaian pertidaksamaan di atas adalah:
-8 ≤ 2/5 (k – 2)
↔ -8 × 5/2 ≤ k – 2
↔ -20 ≤ k – 2
↔ -20 + 2 ≤ k
↔ -18 ≤ k
↔ k ≥ -18
Kita peroleh himpunan penyelesaian, Hp = {k ≥ -18, k ∈ bilangan real}.
Jika digambarkan dengan garis bilangan, maka himpunan penyelesaian tersebut akan terwakili oleh sebuah tanda panah yang bermula di titik k = -18 dengan bulatan tertutup menuju kea rah kanan dari titik k = -18.
Bulatan tertutup menunjukkan bahwa nilai k = -18 termasuk ke dalam himpunan penyelesaian.
Gambar garis bilangan terlampir.

4) Himpunan penyelesaian dari |2x + 1| ≤ |x - 3|, untuk x bilangan real adalah Hp : { -4 ≤ x ≤ [tex]\bold{\frac{2}{3}}[/tex] }

Pembahasan

Nilai mutlak adalah suatu bilangan riil x, dinyatakan oleh |x|, didefinisikan sebagai

[tex]\bold{|x|}\left \{ {{\bold{x, \: jika \: x \geq 0 } } \atop {\bold{-x, \: jika \: x < 0}}} \right.[/tex]

Misal :

|6| = 6

|0| = 0

|-5| = -(-5) = 5

Sehingga |x| tidak pernah negatif .

.

Persamaan Nilai Mutlak adalah persamaan yang memuat variabel dalam tanda mutlak

Pertidaksamaan Nilai Mutlak adalah pertidaksamaan yang memuat variabel dalam Tanda mutlak

Sifat Pertidaksamaan Nilai Mutlak

1) |f(x)| < a ⇔ -a < f(x) < a

2) |f(x)| ≤ a ⇔ -a ≤ f(x) < a

3) |f(x)| > a ⇔ f(x) < -a atau f(x) > a

4) |f(x)| ≥ a ⇔ f(x) ≤ -a atau f(x) ≥ a

5) |x| = √(x²) ⇔ |x²| = x²

6) |f(x)| < |g(x)| ⇔ [f(x)]² < [g(x)]²

7) |f(x)| ≤ |g(x)| ⇔ [f(x)]² ≤ [g(x)]²

8) |f(x)| > |g(x)| ⇔ [f(x)]² > [g(x)]²

9) |f(x)| ≥ |g(x)| ⇔ [f(x)]² ≥ [g(x)]²

..

Dari penjelasan tersebut, mari kita selesaikan soal berikut.

Diketahui :

1) |2x - 1| = 3x + 11

2) |4x - 2| ≤ 14

Ditanya :

Himpunan Penyelesaian ?

Jawaban :

1) |2x - 1| = 3x + 11

Definisi Nilai Mutlak :

[tex]\bold{|2x - 1|\left \{ {{2x - 1, \: jika \: x \geq \frac{1}{2} } \atop {-(2x-1), \: jika \: x < \frac{1}{2} }} \right. }[/tex]

Untuk x ≥ 0

2x - 1 = 3x + 11

⇔ 2x - 3x = 11 + 1

⇔ -x = 12

⇔ x = -12  → [ Tidak Memenuhi ]

Untuk x < 0

2x - 1 = -(3x + 11)

⇔ 2x - 1 = -3x - 11

⇔ 2x + 3x = -11 + 1

⇔ 5x = -10

⇔ x = [tex]-\frac{10}{5}[/tex]

⇔ x = -2

Jadi, Himpunan penyelesaian dari persamaan nilai mutlak tersebut adalah Hp : { -12, -2 }

..

2) |4x - 2| ≤ 14

Gunakan sifat pertidaksamaan nilai mutlak ke - 2

[tex]\boxed{\bold{|f(x)| \leq a \Leftrightarrow -a \leq f(x) < a}}[/tex]

|4x - 2| ≤ 14

⇔ -14 ≤ 4x - 2 < 14

⇔ -14 + 2 ≤ 4x - 2 + 2 < 14 + 2

⇔ -12 ≤ 4x < 16

⇔ [tex]-\frac{12}{4} \leq x < \frac{16}{4}[/tex]

⇔ -3 ≤ x < 4

Jadi, Himpunan penyelesaian dari pertidaksamaan nilai mutlak tersebut adalah Hp : { -3 ≤ x < 4 }

..

3) |3x + 8| = |x + 1|

Definisi Nilai Mutlak :

[tex]\bold{|3x + 8|\left \{ {{3x + 8,\: jika \: x \geq -\frac{8}{3} ... (1) } \atop {-(3x + 8), \: jika \: x \: < -\frac{8}{3} ... (2) }} \right. }[/tex][tex]\bold{|x + 1|\left \{ {{x + 1,\: jika \: x \geq -1 ... (3) } \atop {-(x + 1), \: jika \: x \: < -1 ... (4) }} \right. }[/tex]

- Untuk persamaan (1) dan (3)

3x + 8 = x + 1

⇔ 3x - x = 1 - 8

⇔ 2x = -7

⇔ x = [tex]-\frac{7}{2}[/tex]    [Tidak Memenuhi]

- Untuk persamaan (1) dan (4)

3x + 8 = -(x + 1)

⇔ 3x + 8 = -x - 1

⇔ 3x + x = -1 - 8

⇔ 4x = -9

⇔ x = [tex]-\frac{9}{4}[/tex]   [Memenuhi]

- Untuk persamaan (2) dan (3)

-(3x + 8) = x + 1

⇔ -3x - 8 = x + 1

⇔ -3x - x = 1 + 8

⇔ -4x = 9

⇔ x = [tex]-\frac{9}{4}[/tex]   [Tidak Memenuhi]

- Untuk persamaan (2) dan (4)

-(3x + 8) = -(x + 1)

⇔ -3x - 8 = -x - 1

⇔ -3x + x = -1 + 8

⇔ -2x = 7

⇔ x = [tex]-\frac{7}{2}[/tex]   [Memenuhi]

Jadi, Himpunan penyelesaian dari persamaan nilai mutlak tersebut adalah Hp : { [tex]\bold{ -\frac{7}{2}, \: -\frac{9}{4} }[/tex] }

..

4) |2x + 1| ≤ |x - 3|

Gunakan sifat pertidaksamaan yang ke 7

[tex]\boxed{\bold{|f(x)| \leq |g(x)| \Leftrightarrow [f(x)]^{2} \leq [g(x)]^{2} }}[/tex]

|2x + 1| ≤ |x - 3|

⇔ [2x + 1]² ≤ [x - 3]²

⇔ [2x + 1]² - [x - 3] ≤ 0

⇔ [2x + 1 + x - 3][2x + 1 - x + 3] ≤ 0

⇔ [3x - 2][x + 4] ≤ 0

Pembuat Nol :

3x - 2 = 0 ⇔ x = [tex]\frac{2}{3}[/tex]x + 4 = 0 ⇔ x = -4
