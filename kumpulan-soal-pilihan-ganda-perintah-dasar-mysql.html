1. perintah perintah mysql


Perintah Mysql :
Menampilkan semua database : mysql> show database;
membuat database baru : mysql> create database mahasiswa; (itu contoh)
Memilih database yang akan digunakan : mysql> use mahasiswa;
membuat tabel : mysql> create table mahasiswa
- > (nopeg int unsigned auto_increment primary key,
- > nama varchar(5) not null,
- > gender varchar(2),
- > alamat varchar(20),
- > kota varchar(15),
- > tanggal lahir date null default '0000-00-00');
untuk melihat tabel : mysql> show tables;
menampilkan struktur tabel : mysql show columns from mahasiswa;
Mengisi data : mysql> insert into mahasiswa
> (nma , gender , alamat , kota , tanggal lahir , kodepos),
> values
> ('Ahmad','LK','Pasar minggu no 2','jakarta','1999-09-24','4152')
> ;
mengisi data (cara 2) : mysql> insert into mahasiswa
> set nama - 'budi',
>gender - 'LK',
>alamat - 'sukabumi no 1',
> kota - 'sukabumi';
>tanggal lahir - '1999-09-24',
Mengisi data (Cara ketiga) : mysql> insert into mahasiswa
> values
>(null,'andl',LK','blok A','Sukabumi','1999-09-24','4152',)
>;
menampilkan data : mysql> select * from mahasiswa
>;
Menambahkan kolom : mysql-> alter table mahasiswa
-> add kode pos varchar(5)
->;
Menghapus database : mysql->drop database mahasiswa;
menghapus kolom : (contoh menghapus kolom alamat)
mysql>alter mahasiswa drop alamat;
Mengubah kolom : mysql-> alter table mahasiswa chage kodepos telepon int(10);

menampilkan kolom pilihan , cth : menampilkan kolom nama dan alamat : mysql->select nama , alamat from mahasiswa;
untuk menampilkan field sesuai abjad : mysql>select * from mahasiswa order by nama;

2. Perintah perintah mysql di cmd


pertama buka mysql dlu dari cmd, haru sesuai lokasi pada hdd
ex: cd C:\xampp\mysql\bin

kemudian jalankan mysql dengan perintah "mysql -u root"

1. Membuat Database: create database 'namadatabase';
   ex: create database sekolah;

2. Menghapus Database: drop database 'namadatabase';
   ex: drop database sekolah;

3. Menampilkan Daftar Database: show databases;

4. Menggunakan/Mengaktifkan Database: use 'namadatabase';
     ex: use sekolah;

5. Membuat Table :
CREATE TABLE nama_tbl(
kolom1 tipekolom1(ukuran),
kolom2 tipekolom2(ukuran),
...
);
ex:
CREATE TABLE Siswa(
NIS Char(10) NOT NULL PRIMARY KEY,
NAMA Varchar(40),
Umur Int Not Null Default 0,
Kelamin Enum('Pria','Wanita')
);
*Jangan Lupa Tanda "," Koma Pada Akhir

6. Melihat Daftar Tabel: show tables;

7. Melihat Deskripsi Tabel: desc 'namatabel';
   ex: desc siswa;

8.Memodifikasi Struktur Tabel
ALTER TABLE nama_tbl action sedangkan perintah untuk menambah kolom baru dengan tipe dan posisi tertentu adalah:
ALTER TABLE nama_tbl
ADD kolom_baru type() definisi

Adapun kueri yang mengandung klausa Group By disebut kueri terkelompok dan hanya mengembalikan satu baris tunggal untuk setiap item yang dikelompokkan.

25. Bagaimana user melakukan pengaksesan dalam MySQL?


Dengan melalui Command promt untuk membuka mysql dan juga dapat melalui Browser untuk membuka mysql yaitu dengan mengetikkan localhost/phpmyadmin pada address Bar

Pembhasan:

MySQL adalah sebuah DBMS (database management system) yang menggunakan perintah SQL (Structured Query Language) yang banyak digunakan saat ini untuk membangun aplikasi berbasis web. MySQL dibagi menjadi dua lisensi, yang pertama adalah perangkat lunak bebas yang memungkinkan siapa saja untuk mengakses perangkat lunak. Yang kedua adalah shareware, perangkat lunak berpemilik dengan batasan penggunaan.

MySQL adalah bagian dari RDBMS (Relational Database Management System). Jadi gunakan tabel, kolom, dan baris dalam struktur database Anda. Jadi kami sedang dalam proses pengumpulan data menggunakan metode database relasional. Juga, koneksi antara perangkat lunak dan server database.

Fungsi

Pada tingkat tinggi, fungsi MySQL adalah menggunakan bahasa SQL untuk membuat dan mengelola database yang menyimpan berbagai informasi di sisi server. Fungsi lainnya adalah memberikan kemudahan bagi pengguna untuk mengakses data yang berisi informasi dalam bentuk string (teks) yang dapat diakses secara pribadi atau publik di Internet.

Hampir semua vendor web server atau web host menawarkan fungsionalitas MySQL untuk mengembangkan aplikasi berbasis web yang dikelola oleh pengembang web. Kedua, antarmuka MySQL adalah PHPMyAdmin. Hal ini untuk menghubungkan bahasa pemrograman PHP ke MySQL untuk proses manajemen database di web.

30. MemasUkkan perintah untuk mengakses database mysql server melalui


Jawaban:

Buka command prompt atau terminal pada komputer Anda.Masuk ke direktori instalasi MySQL Server.Masuk ke folder "bin" pada direktori MySQL Server.Jalankan perintah "mysql -u [username] -p [password]".

         a. Ganti [username] dengan username MySQL Anda.

         b. Ganti [password] dengan password MySQL Anda.

Setelah Anda menjalankan perintah di atas, Anda akan diarahkan ke shell MySQL.Selanjutnya, Anda dapat menggunakan perintah SQL untuk melakukan berbagai operasi seperti membuat database, membuat tabel, mengisi data, dan sebagainya.

Penjelasan:

Berikut adalah contoh perintah untuk membuat database baru:

CREATE DATABASE nama_database;

Berikut adalah contoh perintah untuk memilih database yang sudah ada:

USE nama_database;

Berikut adalah contoh perintah untuk menampilkan tabel yang sudah ada dalam database:

SHOW TABLES;
