Berdasarkan tabel itu, maka rumus fungsi [tex]f[/tex] adalah:
[tex]\boxed{\,f(x)=2x+1\,}[/tex]
 

Penjelasan

Cara 1
Barisan Aritmatika

Karena nilai-nilai [tex]x[/tex] membentuk barisan aritmatika 6 bilangan asli pertama, maka kita dapat menganggap nilai-nilai [tex]f(x)[/tex] sebagai barisan aritmatika juga. Perhatikan bahwa beda/selisih antara setiap nilai [tex]f(x)[/tex] yang berurutan adalah tetap, yaitu 2.

[tex]\begin{aligned}U_x&=a+(x-1)b\\&\quad(a=3,\ b=2)\\U_x&=3+(x-1)(2)\\&=3+2x-2\\&=2x+1\\U_x&=f(x)=2x+1\end{aligned}[/tex]

Cara 2
Membentuk SPLDV

Dari tabel argumen dan nilai fungsi, dengan [tex]f(x)=ax+b[/tex] kita peroleh:

[tex]f(1) = 3 \iff a+b=3[/tex][tex]f(2) = 5 \iff 2a+b=5[/tex]

Jika fungsi [tex]f(x)[/tex] konsisten, kita tidak memerlukan persamaan yang lain, karena hanya 2 variabel yang akan dicari nilainya.

[tex]\begin{aligned}2a + b &= 5\\a+(a+b)&=5\\a+3&=5\\\bullet\ \:a&=2\\\bullet\ \;b&=(a+b)-a\\&=3-2\\&=1\end{aligned}[/tex]

Dengan [tex]a = 2[/tex] dan [tex]b = 1[/tex], kita peroleh:
[tex]f(x)=2x+1[/tex]

Pemeriksaan

f(1) = 2·1 + 1 = 3f(2) = 2·2 + 1 = 5f(3) = 2·3 + 1 = 7f(4) = 2·4 + 1 = 9f(5) = 2·5 + 1 = 11f(6) = 2·6 + 1 = 13

Semuanya sesuai dengan data pada tabel yang diberikan.

KESIMPULAN

5. ~MathJawab soal dengan lengkap! ​


Diberikan persamaan linear:a. y + 2x = 6b. 3y + 4x - 5 = 0Kemudian, tentukan pasangan berurutan dengan tabel dan gambarkan grafik persamaan fungsi tersebut!

______________________________

JAWABAN:

a. (3, 0) dan (0, 6), tabel dan grafik fungsi terlampir.b. (5/4, 0) dan (0, 5/3), tabel dan grafik fungsi terlampir.Gambar secara berurut = { Tabel a, Tabel b, Grafik Fungsi a, Grafik Fungsi b, Grafik Fungsi Gabungan a dan b }

PENJELASAN:

⇔ Sistem Persamaan Linear Dua Variabel (SPLDV) adalah sistem persamaan linear yang hanya memiliki/memuat dua variabel dalam ruas dengan pangkat satu.

⇔ SPLDV dapat diselesaikan dengan metode substitusi, eliminasi, grafik, Gauss-Jordan (matriks) dan gabungan diantaranya.

⇔ Grafik fungsi persamaan linear merupakan grafik fungsi tidak berlimit, namun mempunyai titik potong (intercept) x dan y yang membentang lurus.

⇔ Untuk menyelesaikan grafik persamaan linear dari suatu persamaan, kita dapat mensubstitusikan salah satu variabel (misal x atau y) dengan 0 dan mendapatkan koordinat salah satu variabel. Gunakan cara yang sama untuk variabel yang lain, kemudian lukis dalam grafik.

PEMBAHASAN SOAL:

⇒ a. y + 2x = 6

Terdapat dua variabel disini, ada variabel y dengan koefisien 1 dan x dengan koefisien 2 dengan nilai konstanta 6 di ruas kanan.

Selesaikan dengan mensubstitusikan salah satu variabel dengan 0 dan dapatkan pasangan koordinat (x, y).

→ Substitusikan nilai y = 0:

y + 2x = 60 + 2x = 62x = 62x : 2 = 6 : 2x = 3 dan y = 0(x, y) = (3, 0)

→ Substitusikan nilai x = 0:

y + 2x = 6y + 2(0) = 6y = 6 dan x = 0(x, y) = (0, 6)

Jadi, titik potong horizontal (garis x) adalah (3, 0) dan titik potong (intercept) vertikal (garis y) adalah (0, 6).

→ Tabel dan Grafik Fungsi Linear terlampir.

______________________________

⇒ b. 3y + 4x - 5 = 0

Terdapat dua variabel disini, ada variabel y dengan koefisien 3 dan x dengan koefisien 4 dengan nilai konstanta -5 di ruas kiri dan 0 di ruas kanan.

Selesaikan dengan mensubstitusikan salah satu variabel dengan 0 dan dapatkan pasangan koordinat (x, y).

→ Substitusikan nilai y = 0:

3y + 4x - 5 = 03(0) + 4x - 5 + 5 = 0 + 54x = 54x : 4 = 5 : 4x = 5/4 dan y = 0(x, y) = (5/4, 0)

→ Substitusikan nilai x = 0:

3y + 4x - 5 = 03y + 4(0) - 5 + 5 = 0 + 53y = 53y : 3 = 5 : 3y = 5/3 dan x = 0(x, y) = (0, 5/3)

Jadi, titik potong horizontal (garis x) adalah (5/4, 0) dan titik potong (intercept) vertikal (garis y) adalah (0, 5/3).

→ Tabel dan Grafik Fungsi Linear terlampir.

[tex]U_n=28-2n+2[/tex]

[tex]U_n=28+2-2n[/tex]

[tex]\bf U_n=30-2n[/tex]

Beda → b = 26 - 28 = -2

Rumus suku ke-n

Un = 28 + (n - 1) × (-2)

Un = 28 + (-2n) - (-2)

Un = 28 - 2n + 2

Un = -2n + (28 + 2)

Un = -2n + 30

xy = ... ?

Penyelesaian

x² = 8x + 89

y² = 8x + 89

_________ -

(x² - y²) = 8y - 8x

(x² - y²) = -8(x - y)

(x + y) (x - y) = -8(x - y)

x + y = -8

.

x² - y² = (8y + 89) + (8x + 89)

          = 8(x + y) + 178

          = 8(-8) + 178

          = -64 + 178

          = 114

.

(x + y)² = x² + y² + 2xy

2xy = (x + y)² - (x² + y²)

xy = [tex]\frac{(x+y)^{2}-(x^{2}+y^{2})}{2}[/tex]

    = [tex]\frac{(-8)^{2}-114}{2}[/tex]

    = [tex]\frac{64-114}{2}[/tex]

    = [tex]\frac{-50}{2}[/tex]

Maka, nilai xy adalah -25 (D).

33. mohon di bantu ini soal math saya​


Nilai dari [tex]\frac{a}{b}[/tex] adalah 5.

- Pembahasan -

Persamaan linear dua variabel adalah sebuah persamaan aljabar yang memiliki dua variabel ( huruf abjad ) . Terdapat 4 metode untuk menyelesaikan permasalahan tersebut.

- Metode Penyelesaian -Metode Eliminasi ( menghilangkan salah satu suku ) .Metode Subtitusi ( menggantikan suku - suku tersebut ) .Metode Campuran ( Eliminasi + Subtitusi dan sebaliknya )Metode Grafik ( mencari titik-titik koordinat yang memenuhi persamaan.

Metode yang paling tepat digunakan adalah campuran.

Diketahui : a + b = 60 , a - b = 40

Ditanya : [tex]\frac{a}{b}[/tex] ?

Penyelesaian :

Metode Eliminasi.

[tex]a + b = 60\\a - b = 40\\-----[/tex]

[tex]2b = 20[/tex]

[tex]b = \frac{20}{2}[/tex]

[tex]b = 10[/tex]

Metode Subtitusi ( disubtitusikan ke persamaan 1 ) .

[tex]a + b = 60[/tex]

[tex]a + 10 = 60[/tex]

[tex]a = 60 - 10[/tex]

[tex]a = 50[/tex]

Mencari hasil dari [tex]\frac{a}{b}[/tex]

[tex]= \frac{a}{b}[/tex]

[tex]= \frac{50}{10}[/tex]

[tex]= 5[/tex]

Jadi, hasil dari [tex]\frac{a}{b}[/tex] adalah 5.
