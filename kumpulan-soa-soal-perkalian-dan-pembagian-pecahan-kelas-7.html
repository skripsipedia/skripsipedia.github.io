Berdasarkan konsep bangun ruang, kerangka dapat ditentukan melalui :

Kerangka kubus = 12 x rusuk

= 12 x 7cm

= 84cm

Kerangka balok = 4(p + l + t)

= 4(14 + 7 + 7)

= 4(28)

= 112cm

Kerangka limas = 4 x rusuk alas + 4 x sisi tegak

= 4 x 7 + 4 x 7

= 28 + 28cm

= 56cm

Kerangka prisma segitiga sama sisi

= 6 x rusuk segitiga + 3 x tinggi prisma

= 6 x 7 + 3 x 7

= 7(9)

= 63cm

Dijumlahkan seluruhnya

= 84 + 112 + 56 + 63cm

= 315cm

Kawat yang tersedia = 3,5 meter = 350cm

Kawat yang tersisa = 350 - 315cm = 35cm
