14. Plis bantu jawab yang lengkap plis KIMIA KELAS 11 IPA​


Jawaban:

massa yang CH4 harus dibakar untuk memperoleh kalor sebesar 1000 kJ yaitu 20 gram

Penjelasan:

untuk mengetahui massa yang digunakan, maka harus mengetahui entalphi pembakaran standar dari CH4, kemudian mencari perbandingan molnya. setelah itu massa nya akan ditemukan.

1. Entalpi pembakaran standar dari CH4 = - 802 kJ/mol

2. Mencari perbandingan mol dimana

802 kJ dilepaskan untuk 1 mol

maka jika 100 kJ berapa mol yang  diperlukan?

[tex]\frac{802}{1000} = \frac{1}{x } \\802 . x = 1000\\x = \frac{1000}{802}\\x = 1,25 mol[/tex]

3. Mencari Mr

Mr CH4 = Ar C + (Ar Hx4)

Mr CH4 = 12 + (1x4)

Mr CH4 = 12 + 4

Mr CH4 = 16

4. Merubah mol menjadi massa

Mencari massa yakni mol dikalikan dengan Mr CH4

[tex]massa = mol . Mr\\massa = 1,25 . 16\\massa = 20 gram[/tex]

[tex]\int\limits^4_1 {3\sqrt{x} } \, dx[/tex]

[tex]= 3 \int\limits {\sqrt{x} } \, dx[/tex]

[tex]= 3 \int\limits x^{\frac{1}{2} } \, dx[/tex]

[tex]= 3 . \frac{2x^{\frac{3}{2} } }{3}[/tex]

[tex]= 2x^{\frac{3}{2} }[/tex]

[tex]= 2x\sqrt{x}[/tex]

[tex]= 2x\sqrt{x} I^4_1[/tex]

[tex]= 2. 4\sqrt{4} - 2.1\sqrt{1}[/tex]

[tex]\int\limits^5_2 {\frac{4}{x^{2} } } \, dx[/tex]

[tex]= 4\int\limits {\frac{1}{x^{2} } } \, dx[/tex]

[tex]= 4 . (-\frac{1}{x} )[/tex]

[tex]=-\frac{4}{x}[/tex]

[tex]=-\frac{4}{x} I^5_2[/tex]

[tex]=-\frac{4}{5} -(-\frac{4}{2} )[/tex]

[tex]=-\frac{4}{5} -(-2 )[/tex]

[tex]=-\frac{4}{5} -(-\frac{10}{5} )[/tex]

[tex]=\frac{6}{5}[/tex]

[tex]\int\limits^1_0 {(2x+1)^{3} } \, dx[/tex]

misal 2x + 1 = u

[tex]=\int\limits {(2x+1)^{3} } \, dx[/tex]

[tex]=\int\limits {\frac{u^{3} }{2} } \, du[/tex]

[tex]=\frac{1}{2} \int\limits u^{3} \, du[/tex]

[tex]=\frac{1}{2} . \frac{u^{4} }{4}[/tex]
