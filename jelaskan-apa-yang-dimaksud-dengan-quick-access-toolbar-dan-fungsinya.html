1. jelaskan fungsi quick access toolbar


Quick Access Toolbar terletak di sebelah kanan Office Button pada bagian Title Bar. Komponen ini berfungsi untuk menampilkan command button (tombol-tombol shortcut) yang sering Anda gunakan untuk mengolah dokumen. Sebagai contoh, tool Open merupakan shortcut dari Office Button > Open, atau tool Save merupakan shortcut dari Office Button > Save. Anda bisa menambah dan mengurangi command button yang ditampilkan di Quick.
Tombol panah yang terdapat di sebelah kanannya adalah tombol Customize Quick Access Toolbar. Coba klik tombol panah tersebut untuk melihat menu yang ada di dalamnya.

7. fungsi quick access toolbar pada microsoft access


Fungsi dari Quick Access Toolbar pada microsoft access adalah menampilkan perintah-perintah tertentu agar bisa di akses dengan cepat. Perintah yang dimaksud misalnya perintah save, undo, dan redo.

Kenapa cepat ? Karena saat mengklik ikon perintah di Quick Access Toolbar, kita tidak perlu memilih Office Button terlebih dahulu sebelum menemukan perintah yang diinginkan.

Maksudnya, quick access toolbar memungkinkan kita untuk melakukan sebuah perintah lebih cepat daripada menggunakan Office Button.

19. Fungsi quick access toolbar


Tombol Save, berfungsi untuk menyimpan file dokumen word yang sedang dibuat atau sedang terbuka.

Tombol Undo, berfungsi untuk membatalkan perubahan file dari dokumen ketikan yang sedang dibuat/terbuka.

Tombol Redo, berfungsi sebaliknya dari fungsi tombol undo yaitu membatalkan perintah dari tombol undo.

Tombol E-mail, berfungsi membuka aplikasi outlook exprees untuk mengirim dan menerima surat atau dokumen.

Tombol New, berfungsi untuk membuka dan sekaligus membuat file dokumen Microsoft Word yang baru [membuka dan membuat lembar kerja/file dokumen baru]

Tombol Open, berfungsi untuk membuka file dokumen word yang sudah pernah dibuat dan tersimpan pada media penyimpanan baik internal maupun eksternal.

Tombol Quick Print, berfungsi sebagai tombol akses cepat untuk mencetak/printing dokumen. Anda juga dapat melakukan perintah mencetak file atau dokumen word dengan menekan tombol shortcut keyboard tombol CTRL + P secara bersamaan.

Tombol Print Preview, berfungsi untuk melihat tampilan naskah atau dokumen sebelum file dokumen tersebut di cetak dengan menggunakan printer yang sudah terinstall pada computer.

Tombol Spalling & Grammar, berfungsi untuk memeriksa ejaan dan tata bahasa teks pada file dokumen yang sedang dibuat.

Tombol Drraw Table, berfungsi untuk membuat, menyisipkan atau memasukkan tabel ke dalam file dokumen Word yang sedang dibuat.
