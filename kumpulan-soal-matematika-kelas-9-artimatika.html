Penjelasan dengan langkah-langkah:

Perlu diingat bahwa Rumus Barisan Aritmatika adalah:

   Un = a + ( n – 1 ) b

   b = Un -U(n-1)    atau     b= U(n+1) – Un

   Keterangan :

   Un = suku ke n

   n = banyaknya suku

   a = suku pertama

   b = rasio atau beda

Sedangkan Rumus untuk jumlah aritmatika atau disebut Deret Aritmatika (Sₙ) adalah:

Sn = n/2 (2a + (n-1) b)

   Keterangan :

   Sn = suku ke n

   n = banyaknya suku

   a = suku pertama

   b = rasio atau beda

1. Diketahui : a= 10

                     b= 3

Ditanya : S₁₅=....

Jawab:

Sn = n/2 (2a + (n-1) b)

S₁₅= 15/2 (2(10) + (15-1)3)

   = 7.5 (20 + (14)3)

   = 7.5 (20 + 42)

   = 7.5 (62)

  = 465

2. Diketahui : Barisannya Aritmatika adalah 2,4,8...

Ditanya : S₆=....

Jawab: Untuk cara kedua saya menggunakan cara berbeda dan mungkin bisa lebih mudah dimengerti. gunakan cara ini hanya untuk perhitungan cepat saja.

Barisan awal adalah 2,4,8 maka 3 angka selanjutnya adalah 16,32,64 (tiap-tiap angka berikutnya dikalikan 2)

Jadi S₆=2+4+8+16+32+64= 126

29. diketahui artimatika dengan u2 = 9 dan u6 = 25 suku ke 25 adalah


Jika diketahui barisan aritmetika dengan suku ke-2 adalah 9 dan suku ke-6 adalah 25, maka suku ke-25 dari barisan tersebut adalah 101.

Pembahasan

Barisan aritmetika adalah suatu barisan bilangan dengan setiap suku-sukunya  berurutan dan mempunyai selisih selisih yang tetap. Selisih pada barisan aritmetika disebut beda.

Secara umum, rumus suku ke-n pada barisan aritmetika dapat dituliskan sebagai berikut.

Un = a + (n - 1) b

dengan

Un = suku ke-n

a = suku pertama

b = beda = [tex]U_n - U_{n - 1}[/tex]

Deret aritmetika adalah penjumlahan dari suku-suku barisan aritmetika. Secara umum jumlah n suku pertama dari deret aritmetika dapat dituliskan sebagai berikut.

[tex]S_n = \frac{n}{2} (2a + (n - 1)b) \: atau \: S_n = \frac{n}{2} (a + U_n)[/tex]

dengan

Sn = jumlah n suku pertama

Penyelesaian

diket:

U₂ = 9

U₆ = 25

ditanya:

suku ke-25....?

jawab:

- mencari nilai a dan b terlebih dulu

  U₂ = 9   ----> a + (2 - 1) b = 9

                       a + b = 9 .... persamaan (1)

  U₆ = 25  ---> a + (6 - 1) b = 25

                       a + 5b = 25 .... persamaan (2)

 Eliminasi persamaan (1) dan (2)

 a + b = 9

 a + 5b = 25

 _________ +

      -4b = -16

        b = -16 : -4

        b = 4

 Substitusi b = 4 ke persamaan (1)

 a + b = 9

 a + 4 = 9

 a = 9 - 4

 a = 5

- mencari suku ke-25 ---> n = 25

 Un = a + (n - 1) b

 U₂₅ = 5 + (25 - 1) (4)

        = 5 + (24)(4)

        = 5 + 96

 U₂₅ = 101

Kesimpulan

Jadi, suku ke-25 dari barisan tersebut adalah 101.

32. deret artimatika...........


1. Nilai a (suku pertama) adalah a = 4 dan nilai b (beda) adalah b = 5.

2. Jumlah 10 suku pertama barisan tersebut adalah 265.

Pendahuluan

Soal tersebut adalah soal tentang barisan dan deret aritmatika. Dalam mengerjakan soal sejenis ini, perlu diketahui 2 rumus dasar dalam melakukan perhitungan sebagai berikut :

1. Rumus untuk suku ke-n (Un)

[tex]\boxed {\bold {Un = a + (n-1)b}}[/tex]  

Keterangan :

Un = suku ke-n

a = suku awal

n = nomor suku

b = beda/selisih 2 suku berdekatan

2. Rumus jumlah n suku pertama

[tex]\boxed {\bold {Sn = \frac{n}{2} \times (2a + (n-1)b) }}[/tex]  

atau

[tex]\boxed {\bold {Sn = \frac{n}{2} \times (a + Un)}}[/tex]

Keterangan :

Sn = Jumlah n suku pertama

n = nomor suku

a = suku pertama

b = beda/selisih antar suku

Un = suku ke-n (suku terakhir n suku pertama)

Pembahasan

Diketahui :

U₂ = 9

U₆ = 29

Ditanyakan :

Nilai a (suku pertama) dan b (beda/selisih 2 suku berdekatan).

Jumlah 10 suku pertama.

Jawaban :

Karena U₂ = 9, dengan rumus suku ke-n maka didapat

Un = a + (n-1)b

U₂ = a + (2-1)b

9 = a + b .... (persamaan 1)

Karena U₆ = 29, dengan rumus suku ke-n maka didapat

Un = a + (n-1)b

U₆ = a + (6-1)b

29 = a  + 5b ... (persamaan 2)

Kurangi persamaan 2 dengan persamaan 1, didapat

a + 5b - (a + b) = 29 - 9

4b = 20

b = 20 : 4

b = 5

Karena b = 5, maka dari persamaan 1 didapat

a + b = 9

a + 5 = 9

a = 9 - 5

a = 4

Karena a = 4 dan b = 5, maka dengan rumus jumlah n suku pertama didapat jumlah 10 suku pertama adalah

Sn = [tex]\frac{n}{2}[/tex] x (2a + (n-1)b)

S₁₀ = [tex]\frac{10}{2}[/tex] x (2 x 4 + (10 - 1) x 5)

S₁₀  = 5 x (8 + 9 x 5)

S₁₀  = 5 x (8 + 45)

S₁₀  = 5 x 53

S₁₀  = 265

Jadi, a = 4, b = 5, dan jumlah 10 suku pertama adalah 265.
