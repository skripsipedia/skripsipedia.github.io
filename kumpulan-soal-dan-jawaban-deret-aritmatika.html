1. Tentukan suku yang diminta!

a. Diketahui : U1 = a = 1 dan Selisih = b = 3

Ditanya : U25 = ?

Jawab : U25 = a + (n-1)b

= 1 + (25-1)3

= 1 + 24 × 3

U25 = 1 + 72 = 73

b. Diketahui : U1 = a = (-8) dan b = 4

Ditanya : U32 = ?

Jawab : U32 = a + (n-1)b

= -8 + (32-1)4

= -8 + 31 × 4

U32 = -8 + 124 = 124 - 8 = 116

c. Diketahui : U1 = a = 12 dan b = 2

Ditanya : U15 = ?

Jawab : U15 = a + (n-1)b

= 12 + (15-1)2

= 12 + 14 × 2

U15 = 12 + 28 = 40

2. Tentukan jumlah deret aritmetika berikut!

a. Diketahui : U1 = a =2 dan b = 4

Ditanya : S20 = ?

Jawab : S20 = 1/2 × n(U1 + U20)

= 1/2 × 20(2+(a + (n-1)b))

= 10(2+(2 + 19 × 4))

= 10(2+(2+76))

S20 = 10(2+78) = 10 × 80 = 800

b. Diketahui : U1 = a = (-4) dan b = 2

Ditanya : S18 = ?

Jawab : S18 = 1/2 × n(U1 + U18)

= 1/2 × 18(-4 + (-4 +(18-1)2))

= 9(-4 + (-4 + 17 × 2))

= 9(-4 + (34 - 4))

S18 = 9(-4 + 30) = 9 × 26 = 234

3. Diketahui : U1 = a = 5, b = 3, dan Un = 149

Ditanya : 149 merupakan suku ke-?

Jawab : Un = a +(n-1)b

149 = 5 + (n-1)3

149 = 5 + 3n - 3

149 = 2 + 3n

149 - 2 = 3n

147 ÷ 3 = n

49 = n

Maka, 149 merupakan suku ke-49 dari barisan tsb.

13. 4. dari suatu deret aritmatika diketahui u3 = 13 dan u7 = 29. jumlah dua puluh lima suku pertama deret tersebut adalah....... tolong bantu jawab kak sama cara kerjanya juga kak plis bantu kak soalnya besok kumpul kak


Jawaban:

S25 = 1.325

Pembahasan Materi:

Barisan aritmatika adalah suatu baris yang nilai setiap sukunya didapatkan dari suku sebelumnya melalui penjumlahan atau pengurangan dengan suatu bilangan.

Rumus barisan aritmatika:

Un = U1 + (n - 1) × b

= a + (n - 1) × b

-------

Deret aritmatika adalah suatu penjumlahan suku-suku dari barisan aritmatika.

Rumus deret aritmatika:

Sn = ½ × n × (a + Un)

= ½ × n × (2a + (n - 1) × b)

Keterangan:

Un = suku ke-n

U1 = a = suku pertama

b = beda

n = banyak suku pada barisan aritmatika

Penyelesaian:

Diketahui:

U3 = 13

U7 = 29

Ditanya: Jumlah 25 suku pertama deret aritmatika = ?

Dijawab:

- Mencari nilai beda dan suku pertama.

U3 = a + 2b = 13

U7 = a + 6b = 29 -

----------------------------

-4b = -16

b = 4

a + 2b = 13

a + 2(4) = 13

a + 8 = 13

a = 13 - 8

= 5

- Menghtiung nilai S25.

S25 = ½ × n × (2a + (n - 1) × b)

= ½ × 25 × (2(5) + (25 - 1) × 4)

= ½ × 25 × (10 + 24 × 4)

= ½ × 25 × (10 + 96)

= ½ × 25 × 106

= 25 × 53

= 1.325

Jadi, jumlah 25 suku pertama deret tersebut adalah 1.325.
