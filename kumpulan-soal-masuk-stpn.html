1. Jika STIS=67 maka,(STAN)(AMG)(STPI)/(UI)(STIP) adalah


STIS = 67
S = 19, T = 20, I= 9
STIS = 19+20+9+19=67

STAN = 19+20+1+14 = 54
AMG = 1+13+7= 21
STPI = 19+20+16+9= 64
UI = 21+9 = 30
STIP = 19+20+9+16= 64

maka,(STAN)(AMG)(STPI)/(UI)(STIP)
= 54.21.64 / 30.64
= 54.21 / 30
= 1134/30
= 378/10
= 37,8
