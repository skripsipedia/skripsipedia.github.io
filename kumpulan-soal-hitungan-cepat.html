4. Andi berlari dengan kecepatan 10 m/s menjadi 80 m/s Jika perubahan waktunya 5 sekon hitunglah kecepatan Andi berlaritolong ya kk kk ...tolOng dibantu soalnya mau dikumpulkan besok​


Kita memiliki kecepatan awal (v_awal) = 10 m/s, kecepatan akhir (v_akhir) = 80 m/s, dan perubahan waktu (Δt) = 5 s.

Perubahan kecepatan (Δv) dapat dihitung dengan selisih antara kecepatan akhir dan kecepatan awal:
Δv = v_akhir - v_awal
Δv = 80 m/s - 10 m/s = 70 m/s

Untuk menghitung kecepatan rata-rata, kita dapat menggunakan rumus:
v_rata = Δv / Δt
v_rata = 70 m/s / 5 s = 14 m/s

Jadi, kecepatan rata-rata saat Andi berlari adalah 14 m/s.

1.) Diket :

vo = 0 m/s (diam)

vt = 25 m/s

t = 10 s

Ditanya : a ?

Jawab :

a = (vt - vo) / t

a = (25 -0) / 10

a = 2,5 m/s²

2.) Diket :

m = 40 kg

F1 = 25 N (kekanan)

F2 = 75 N (kekanan)

Ditanya : a ?

Jawab :

a = ∑F / m

a = (F1 + F2) / 40

a = (25 + 75) / 40

a = ⁵/₂ m/s² Atau a = 2,5 m/s²

38. sebuah kereta api mula mula bergerak dengan kecepatan 72 km per jam dalam waktu 4 sekon sehingga kecepatannya berubah 180 km per jam hitunglah jumlah percepatan nya tolong di jawab ya soalnya besok di kumpul plis


• GAYA DAN PERCEPATAN



* Diketahui
Vo = 72 km/jam = 20 m/s
Vt = 180 km/jam = 50 m/s
t = 4 sekon

* Ditanya
a = __?

* Jawab

Karena kereta api mengalami pertambahan kecepatan (percepatan), maka bisa disimpulkan bahwa kereta api mengalami gerak lurus berubah beraturan dipercepat.

Vt = Vo + a.t
50 = 20 + a.4
50 - 20 = 4.a
30/4 = a
a = 7,5 m/s². ←

Jadi, besar dari percepatan kereta api adalah 7,5 m/s².
