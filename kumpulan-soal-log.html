23. himpunan penyelesaian dari pertidaksamaan ⁴log (x - 1) ≤ ¼ log (2x - 1) adalah (Dengan cara)Soalnya besok dikumpulkan jadi tolong ya​


Jawab:

⁴log (x - 1) ≤ ¼ log (2x - 1)

Syarat:

x - 1 > 0 dan (2x - 1) > 0

x > 1 dan x > 1/2

⁴log (x - 1) ≤  ⁴log (2x - 1)^-1

(x - 1) ≤ (2x - 1)^-1

(x - 1) ≤ 1/(2x - 1)

(x - 1)(2x - 1) ≤ 1

2x² - 3x + 1 - 1 ≤ 0

2x² - 3x ≤ 0

x(2x - 3) ≤ 0

0 ≤ x ≤ 3/2, x > 1 dan x > 1/2

sehingga

1 < x ≤ 3/2

HP = { x | 1 < x ≤ 3/2}
