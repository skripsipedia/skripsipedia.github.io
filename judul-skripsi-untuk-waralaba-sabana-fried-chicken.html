7. buatlah Phyton Gerobak Fried chicken​


Contoh kode sederhana dalam Python yang menggambarkan "Gerobak Fried Chicken":

```python

class GerobakFriedChicken:

def __init__(self):

self.chicken_stock = 100 # Jumlah ayam dalam stok

self.oil_stock = 10 # Jumlah minyak dalam stok

self.money_earned = 0 # Uang yang diperoleh dari penjualan

def fry_chicken(self, amount):

if self.chicken_stock >= amount and self.oil_stock >= amount:

self.chicken_stock -= amount

self.oil_stock -= amount

self.money_earned += amount * 10 # Harga per potong ayam

print(f"{amount} potong ayam digoreng.")

else:

print("Stok ayam atau minyak tidak mencukupi untuk menggoreng.")

def restock_chicken(self, amount):

self.chicken_stock += amount

print(f"Stok ayam ditambahkan sebanyak {amount} potong.")

def restock_oil(self, amount):

self.oil_stock += amount

print(f"Stok minyak ditambahkan sebanyak {amount} liter.")

def show_stock_and_earnings(self):

print(f"Stok ayam: {self.chicken_stock} potong")

print(f"Stok minyak: {self.oil_stock} liter")

print(f"Total uang yang diperoleh: Rp {self.money_earned}")

# Membuat objek Gerobak Fried Chicken

gerobak_kfc = GerobakFriedChicken()

# Contoh penggunaan

gerobak_kfc.show_stock_and_earnings()

gerobak_kfc.fry_chicken(5)

gerobak_kfc.show_stock_and_earnings()

gerobak_kfc.restock_chicken(20)

gerobak_kfc.restock_oil(5)

gerobak_kfc.fry_chicken(10)

gerobak_kfc.show_stock_and_earnings()

```

Kode di atas adalah simulasi sederhana dari gerobak penjualan ayam goreng. Elo dapat menambahkan atau mengubah metode dan atribut sesuai kebutuhan Elo. Pastikan untuk menyesuaikan harga, kapasitas stok, dan operasi lain sesuai dengan situasi sebenarnya.

Jawaban:

Passive Voice:

1. Fried chicken is cooked by Mary.

2. Fried chicken is being cooked by Mary.

3. Fried chicken will be cooked by Mary.

4. Fried chicken has been cooked by Mary.

5. Fried chicken was cooked by Mary.

Penjelasan:

Jawaban merupakan materi pembelajaran bentuk kalimat pasif dalam Bahasa Inggris.

Passive Voice:

1. Passive Voice [Simple Present Tense]

-Pattern = Subject + to be (is/am/are) + Verb3

2. Passive Voice [Present Continuous Tense]

-Pattern = Subject + to be (is/am/are) + being + Verb3

3. Passive Voice [Present Perfect Tense]

-Pattern = Subject + have/has + been + Verb3

4. Passive Voice [Simple Past Tense]

-Pattern = Subject + to be (was – were) + Verb 3

5. Passive Voice [Past Continuous Tense]

-Pattern = Subject + to be (was – were) + Being + Verb3

6. Passive Voice [Past Perfect Tense]

-Pattern = Subject + had + been + Verb3

7. Passive Voice [Simple Future Tense]

-Pattern = Subject + will + be + Verb3

8. Passive Voice [Future Continuous Tense]

-Pattern = Subject + will be + being + Verb3

9. Passive Voice [Future Perfect Tense]

-Pattern = Subject + will have + been + Verb3
