1. apa bila sebuah benda bergetar sebanyak 600 getaran setiap 5 menit hitunglah periode getaranya?​


Jawaban:

2 s

Penjelasan:

n = 600

t = 5 menit = 300 s

T = t/n = 600/300 = 2 s

Jadi, periode getarnya adalah 2 s

PEMBAHASAN

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas. Dalam getaran juga terdapat kesetimbangan dan simpangan.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi.

[tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode.

[tex]\bold{n}[/tex] = banyak getaran.

[tex]\bold{t}[/tex] = waktu.

Berarti :

[tex]\boxed{\bold{T= \frac{1}{f}}}[/tex]

waktu = 0,5 menit = 30 detik.

3. banyak getaran = 360 kali.

waktu = 2 menit = 120 detik.

4. banyak getaran = 3.600 kali.

waktu = 3 menit = 180 detik.

Ditanya :

Periode dan frekuensi getaran?

Dijawab :

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{30}{120}}[/tex]

[tex]\bold{T = 0,25 \: s}[/tex]

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{120}{30}}[/tex]

[tex]\bold{f = 4 \: Hz}[/tex]

2. Mencari periode :

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{300}{3.000}}[/tex]

[tex]\bold{T = 0,1 \: s}[/tex]

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{3.000}{300}}[/tex]

[tex]\bold{f = 10 \: Hz}[/tex]

3. Mencari periode :

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{120}{360}}[/tex]

[tex]\bold{T = 0,333 \: s}[/tex]

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{360}{120}}[/tex]

[tex]\bold{f = 3 \: Hz}[/tex]

4. Mencari periode :

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{180}{3.600}}[/tex]

[tex]\bold{T = 0,05 \: s}[/tex]

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{3.600}{180}}[/tex]

[tex]\bold{f = 20 \: Hz}[/tex]

35. Sebuah benda bergetar dengan periode getaranya 0,025 second. berapakah frekuensi dan jumlah getaran selama 5 second?


Jawaban:

jumlah getaran selama 5 detik adalah 200.

Penjelasan:

Frekuensi (f) didefinisikan sebagai jumlah getaran per detik. Untuk menghitung frekuensi, kita dapat menggunakan rumus:

f = 1 / T

di mana T adalah periode getaran.

Dalam kasus ini, periode getaran (T) adalah 0,025 detik. Mari kita gunakan rumus tersebut untuk menghitung frekuensi:

f = 1 / 0,025 = 40 Hz

Jadi, frekuensi getaran tersebut adalah 40 Hz.

Jumlah getaran (N) selama 5 detik dapat dihitung dengan menggunakan rumus:

N = f * t

di mana f adalah frekuensi dan t adalah waktu.

Dalam kasus ini, frekuensi (f) adalah 40 Hz dan waktu (t) adalah 5 detik. Mari kita gunakan rumus tersebut untuk menghitung jumlah getaran:

N = 40 * 5 = 200

Jadi, jumlah getaran selama 5 detik adalah 200.
