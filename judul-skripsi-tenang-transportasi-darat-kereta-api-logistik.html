9. Kereta api termasuk alat transportasi darat.Tulislah argumen berdasarkan kalimat di atas beserta alasannya!​


Teks Argumen tentang Kereta Api adalah:

Transportasi kereta api mempunyai keunggulan dibandingkan dengan transportasi darat yang lain. Keunggulan itu antara lain hemat energi dan dampak lingkungan yang ditimbulkan relatif lebih kecil. Selain itu kereta api merupakan sarana angkutan yang lebih ekonomis baik bagi penumpang maupun barang untuk jarak jauh. Hal ini diungkapkan oleh kepala PJKA dalam harian ibukota.

34. Kegiatan transportasi darat (kereta api) di indonesia dimulai pada masa kolonial


Kegiatan transportasi darat khususnya kereta api di Indonesia dimulai pada masa KOLONIAL BELANDA. Saat itu Indonesia belum merdeka dan lebih dikenal dengan sebutan HINDIA BELANDA. Pembangunan jalur kereta api sendiri dimulai saat Gubernur Jenderal Hindia Belanda Tuan Baron van Den Beele berkuasa.

Pembahasan

Adapun pembangunan transportasi kereta api di Hindia Belanda tersebut dilaksanakan oleh sebuah perusahaan bernama N.V. NISM (singkatan dari Naamlooze Vennootschap Nederlands(ch)-Indische Spoorweg Maatschappij). Perusahaan ini merupakan badan usaha yang bergerak dalam pengadaan kereta api dari Belanda. Kantor dari perusahaan N.V. NISM ini sendiri berlokasi di Semarang. Bangunan kantor tersebut masih tersohor hingga saat ini namun lebih dikenal dengan nama Lawang Sewu.
