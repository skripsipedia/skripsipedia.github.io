Penjelasan:

Impuls = Gaya (N) x waktu (t)

Satuan Gaya = N = kg.m/s2

maka,

Dimensi Gaya = [M][L][T ⁻²]

Dimensi Waktu = [T]

34. apa rumus berat jenis beserta di mensi mohon bantuannya soalnya pr nya bsk di kumpul


Untuk menuliskan dimensi besaran turunan, kita harus mengetahui dimensi besaran pokok terlebih dahulu. 
Langkah selanjutnya yaitu kita menganalisa besaran turunan itu berasal dari gabungan besaran pokok apa. 
Setelah itu kita bisa menentukan dimensi-nya. 

Cara menganalisa besaran turunan (contoh: Berat-jenis): 
Berat-jenis = Berat per Volume 
Berat = Massa x Gravitasi 
Vulume = Panjang x Panjang x Panjang 
Gravitasi = Kecepatan per Waktu 
Kecepatan = Panjang per Waktu 

*Ingat Berat adalah Gaya, dan Gravitasi adalah Percepatan. 

Jadi besaran Berat-jenis dihasilkan dari gabungan besaran pokok: Massa [M], Panjang [L], dan Waktu [T]. 

Dimensi besaran-nya: 
Massa: [M] satuan kilogram (kg) 
Panjang: [L] satuan meter (m) 
Waktu: [T] satuan sekon (s) 
Kecepatan:[L/T] satuan meter per sekon (m/s) 
Gravitasi: [L/T^2] satuan meter per sekon pangkat dua (m/s^2) 
Berat: [ML/T^2] satuan Newton (N) 
Volume: [L^3] satuan meter pangkat tiga (m^3) 

Jadi dimensi Berat-jenis yaitu: 
[ML/T^2] / [L^3] = [M/T^2L^2] {Massa per Waktu pangkat dua Panjang pangkkat dua} 
Satuannya: kilogram per meter pangkat dua sekon pangkat dua (kg/m^2s^2). 
*Karena satuan internasional untuk Berat adalah Newton (N) dan Volume adalah meter pangkat tiga, maka satuan internasional Berat-jenis menjadi:Newton per meter pangkat tiga (N/m^3) rumus berat jenis

 BJ=berat/volume = N/m^3

dengan dimensi N --> [tex] \frac{ML}{ T^{2} } [/tex]
dan dimensi m -->  L

jadi  [tex] \frac{ML}{ T^{2} } [/tex] / L^3

satuan bj = kg/m^2s^2

= [tex] \frac{M}{ L^{2} T^{2} } [/tex]
