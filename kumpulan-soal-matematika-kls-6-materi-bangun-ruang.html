Soal 1

Volume gabungan

= Volume kerucut + Volume tabung

= ( Luas alas × t ) + ( ⅓ × L. alas × t )

= ( ( π × r² ) × t ) + ( ⅓ × ( π × r² ) × t )

= ( ( 22/7 × 7² ) × 12 ) + ( ⅓ × ( 22/7 × 7² ) × 6 )

= ( 154 × 12 ) + ( ⅓ × 924 )

= 1.848 cm³ + 308 cm³

= 2.156 cm³

Soal 2

Volume balok

= Luas alas × t

= ( P × L ) × t

= ( 10 × 5 ) × 4

= 50 × 4

= 200 cm³

Jawab:

1606 [tex]cm^{2}[/tex]

Penjelasan dengan langkah-langkah:

Lp. Gabungan = L. Tabung Tanpa Tutup + Luas Kerucut tanpa alas

-------------------------------------------------------------

Lp.Tabung tanpa tutup =  L.alas + L.Selimut

= ( [tex]\pi[/tex] x [tex]r^{2}[/tex] ) + ( [tex]\pi[/tex] x d x t )

= ( [tex]\frac{22}{7}[/tex] x [tex](\frac{d}{2} )^{2}[/tex] ) + ( [tex]\frac{22}{7}[/tex] x 14 x 28 )

 = ( [tex]\frac{22}{7}[/tex] x [tex](\frac{14}{2} )^{2}[/tex] ) +  (22 x 2 x 28)

=  ( [tex]\frac{22}{7}[/tex] x [tex]7^{2}[/tex] ) + 1232

= ( 22 x 7 )  + 1232

= 154 + 1232 = 1386 [tex]cm^{2}[/tex]

-----------------------------------------------------------

Lp. Kerucut tanpa alas = [tex]\pi[/tex] x r x s

= [tex]\frac{22}{7}[/tex] x 7 x 10

= 220 [tex]cm^{2}[/tex]

-----------------------------------------------------------

Lp. Gabungan = L. Tabung Tanpa Tutup + Luas Kerucut tanpa alas

Lp. Gabungan = 1386 + 220

= 1606 [tex]cm^{2}[/tex]
