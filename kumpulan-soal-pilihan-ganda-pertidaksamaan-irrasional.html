35. Tuliskan definisi pertidaksamaan rasional dan irrasional.


Pertidaksamaan rasional adalah bentuk pertidaksamaan yang mengandung bilangan-bilangan rasional, seperti pertidaksamaan kuadrat, pertidakdamaan pecahan, dsb.
Contoh:
4x – 8 ≥ 5x + 4
(x – 2)/3 + (2x – 1)/2 > x – 4 
Pertidaksamaan irrasional adalah pertidaksamaan yang mengandung bilangan irrasional, atau lebih mudah dikenali sebagai pertidaksamaan yang mengadung bentuk akar. 
Contoh:
√(x + 6) ≤ x
x – 3 < √(x – 1)


36. Hasil dari pertidaksamaan irrasional dari soal ini adalah?√x²-3x+2 < 2√3​


Penjelasan dengan langkah-langkah:

√(x² - 3x + 2) < 2√3

√(x² - 3x + 2) < √12

x² - 3x + 2 < 12

x² - 3x + 2 - 12 < 0

x² - 3x - 10 < 0

(x + 2)(x - 5) < 0

• Pembuat nol :

x = -2 atau x = 5

• Garis bilangannya :

+++ (-2) ––– (5) +++

karena bertanda "<", berarti arahnya ke negatif. HP1 = {x | -2 < x < 5}

Untuk syarat dari akarnya :

x² - 3x + 2 ≥ 0

(x - 1)(x - 2) ≥ 0

• Pembuat nol :

x = 1 atau x = 2

• Garis bilangannya :

+++ (1) ––– (2) +++

karena bertanda "≥", berarti arahnya ke positif. HP2 = {x | x ≤ 1 atau x ≥ 2}

HP1​ ∩ HP2 = {x | -2 < x ≤ 1 atau 2 ≤ x < 5}

Semoga Bermanfaat
