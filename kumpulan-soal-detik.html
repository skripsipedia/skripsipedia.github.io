Masukkan waktu;

jam=sekon/3600;

menit=(sekon-(3600*(sekon/3600)))/60;

detik=((sekon-(3600*(sekon/3600)))-(((sekon-(3600*(sekon/3600)))/60)*(60)));

tulis konversi waktu;

end;

Implementasi dalam bahasa C++

#include <iostream.h>

#include <conio.h>

class Konversi{

friend istream& operator>>(istream&, Konversi&);

friend ostream& operator<<(ostream&, Konversi&);

public:

Konversi(){};

int j();

int m();

int d();

private:

int jam;

int menit;

int detik;

int sekon;

};

int Konversi :: j(){

jam=sekon/3600;

return jam;

}  

int Konversi :: m(){

menit=(sekon-(3600*(sekon/3600)))/60;

return menit;

}

int Konversi :: d(){

detik=((sekon-(3600*(sekon/3600)))-(((sekon-(3600*(sekon/3600)))/60)*(60)));

return detik;

}

Jawaban:

1dm3/detik

Penjelasan dengan langkah-langkah:

60.000dm3 ke dm3 dibagi 1000 karena satuannya kubik(3)maka hasilnya 60 terus 60:60(60 dari 1 menit dijadikan detik) =1

Jawaban:

Dari persoalan di atas dapat disimpulkan bahwa hasil dari 60.000 cm3/ menit = 1 dm3/detik.

Penjelasan dengan langkah-langkah:

Berdasarkan satuannya dari terbesar sampai terkecil berturut-turut adalah:

Km (kilometer)Hm (hektometer)Dam (dekameter)M (meter)Dm (decimeter)Cm (centimeter)Mm (milimeter)

Perlu diingat bahwa aturan dari satuan tersebut adalah mengikuti pada setiap berapa tingkat satuan tersebut turun atau naik. Atau sama dengan setiap nol pada angka 1 akan berubah sesuai perintah soal.

Jika naik 1 tingkat, maka dibagi 10. Misal: Dam ke Hm (naik satu tingkat). Apabila naik dua tingkat maka dibagi 100. Jika turun satu tingkat, maka dikalikan 10. Misal: Dam ke Cm (turun satu tingkat). Apabila turun dua tingkat, maka dikalikan 100.

Dari ketentuan tersebut diperoleh pengoperasian sebagai berikut:

*1 menit = 60 detik

60.000 cm³/menit = 60.000/1.000 = 60 dm³

60 dm³/60 detik = 1 dm³/detik.
