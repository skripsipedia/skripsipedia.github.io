32. Persamaan kuadrat: 4x2 + 2x – 6 = 0 Hitung akar persaman kuadratnya!


4x2 + 2x - 6 = 0
2(2x2 + x - 3) = 0
2(2x + 3)(x - 1) = 0

Akar persamaannya adalah :
x = -3/2 dan x = 1

Jawab:

4x² + 2x – 6 = 0

4x² - 4x + 6x - 6 = 0

4x(x - 1) + 6(x - 1) = 0

(4x + 6)(x - 1) = 0

4x + 6 = 0

      4x = -6

        x₁ = -6/4 = -1 1/2

x - 1 = 0

   x₂ = 1

Jadi, akar-akar persamaan kuadratnya adalah x₁ = -1 1/2 dan x₂ = 1
