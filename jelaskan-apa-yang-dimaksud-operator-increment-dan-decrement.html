1. jelaskan tentang operator increment dan decrement


Intcrement digunakan untuk pengulangan yang nilainya bertambah/naik ke atas.
Penulisan: a++, a+=1, x+=5

Sedangkan decrement digunakan untuk pengulangan yang nilainya semakin berkurang/turun ke bawah.
Penulisan: x--, a-=1, x-=10

3. fungsi operator Increment Dan Decrement, dalam bahasa c++....tolong!!!!


Increment adalah operator yang digunakan untuk menambah nilai sebuah variable dengan 1 angka dan decrement digunakan untuk mengurangi nilai sebuah variable 1 angka. Contoh penggunaan increment dan decrement dalam C++:


int a = 10;

++a;

printf("%d\n",a); //Hasilnya menjadi 11

printf("%d\n",a++); //Hasilnya menjadi 11 karena nilai diprint terlebih dahulu lalu  ditambah 1 (Post-increment)

printf("%d\n",a); //Hasilnya menjadi 12 setelah diincrement

a--;

printf("%d",a); //Hasilnya menjadi 11

Penjelasan:

Operator adalah simbol yang kita gunakan untuk memerintahkan compiler untuk melakukan operasi tertentu pada dua buah objek. Operasi dimaksud bisa berupa operasi matematis, relasional, ataupun operasi logika. Sebuah statement berupa operasi antara dua buah objek disebut ekspresi.Contoh :

jumlah = 7 + 3

Pada contoh di atas, simbol + adalah operator, sementara 7 + 3 disebut sebagai ekspresi.

Berikut adalah penjelasan mengenai berbagai operator dalam bahasa pemrograman python

Operator matematis (arimatika)

Operator +

Operator + (plus) digunakan untuk menjumlahkan dua objek. Bila kedua objek bernilai numerik maka ia akan dijumlahkan secara numeris. Bila ia berupa teks (string), maka kedua objek tersebut akan digabung.

a = 5

b = 4

c = a + b # c akan bernilai bilangan bulat 9

a = 'Hello'

b = 'World'

c = a +' '+ b # c akan bernilai teks 'Hello World'

Operator –

Operator – (minus) digunakan untuk mengurangi objek dikiri dengan objek di kanan (seperti ekspresi matematis biasa). Jika objek dikiri tidak ada, ia dianggap 0. Operator – tidak dapat digunakan pada nilai teks (string)

a = 5

c = a - 2 # c akan terisi nilai 3

c = - a # c akan terisi nilai -5

Operator * dan **

Operator * (perkalian) digunakan untuk mengalikan dua buah objek, sedangkan operator ** (power) digunakan untuk memangkatkan objek.

a = 3

b = 2

c = a * b # c akan terisi nilai 6

d = a ** b # d akan terisi nilai 9

Operator /, //, dan %

Operator / (pembagi) digunakan untuk membagi objek. Jika yang digunakan adalah operator // maka pembagian akan dilakukan dengan pembulatan (floor). Operator % (modulus) akan menghasilkan sisa dari hasil pembagian.

Perhatikan contoh kode di bawah untuk lebih jelasnya.

a = 7

b = 3

c = a / b # c akan terisi nilai 2.3333

d = a // b # d akan terisi nilai 2

e = a % b # e akan terisi nilai 1

Operator relasional

Operator relasional adalah operator yang kita gunakan untuk menguji nilai kebenaran sebuah ekspresi, kebenaran berarti apakah sebuah ekspresi bernilai True atau False.

Berikut adalah daftar operator relasional dan maknanya:

Operator <, bermakna lebih kecil dari

Operator <=, bermakna lebih kecil atau sama dengan dari

Operator >, bermakna lebih besar dari

Operator >=, bermakna lebih besar atau sama dengan dari

Operator ==, bermakna sama dengan

Operator !=, bermakna tidak sama dengan

Dalam kode berikut disertakan contoh penggunaan operasional beserta hasil ekspresinya.

a = 7

b = 3

c = a < b # c akan terisi False

c = a <= b # c akan terisi False

c = a > b # c akan terisi True

c = a > b # c akan terisi True

c = a == b # c akan terisi False

c = a != b # c akan terisi True

14. tuliskan contoh satu program bahasa c++ yang didalamnya terdapat identifier, konstanta, variabel, tipe data, dan dalam satu program ada operator unary (increment dan decrement), tunjukkan keterangan nya. mohon bantuannya


Soal :

tuliskan contoh satu program bahasa c++ yang didalamnya terdapat identifier, konstanta, variabel, tipe data, dan dalam satu program ada operator unary (increment dan decrement), tunjukkan keterangan nya. mohon bantuannya


Jawaban :

#include <iostream>

using namespace std;

int main( ){

   const int n=9;  //konstanta

   int x=5;  //variabel = x. tipe data = int

   x++;  //Increment

   cout << "Increment : " << (x*n) << endl;

   x--; //Decrement

   cout << "Decrement : " << (x*n) << endl;

   return 0;

}


Pembahasan :

Identifier terdiri atas konstanta dan variabel.

Konstanta bersifat tetap, tidak  berubah meskipun pada program terdapat operasi unary didalamnya. Pada program yang saya buat, konstanta nya adalah n=9 dengan tipe data bilangan bulat (int). Variabel digunakan untuk mewakili suatu nilai tertentu pada program. nilai tertentu tersebut tipe data. Pada program yang saya buat, variabel nya adalah x dengan tipe data bilangan bulat (int).

Tipe data terdiri dari bilangan bulat (short, unsigned short, int, unsigned int, long long, dan unsigned long long), bilangan riil (double dan float), karakter (char) dan boolean (true dan false).

Operator unary terdiri atas increment dan decrement. Pada program yang saya buat, operator incrementnya adalah x++ dan operator decrementnya adalah x--.

Output pada program tersebut ada pada gambar.

Pada output program tersebut, nilai increment nya menjadi 54 karena nilai x yang awal nya 5 menjadi 6 karena mendapat operator unary incrementnya, lalu nilai x setelah mendapat operator unary dikalikan dengan konstanta n yang bernilai 9. Sehingga output menjadi Increment : 54.

Karena nilai x sekarang 6, setelah mendapat operator unary decrement menjadi 5. Lalu nilai x setelah mendapat operator unary dikalikan dengan konstanta n yang bernilai 9. Sehingga output nya menjadi Decrement : 45

33. tuliskan contoh satu program bahasa c++ yang didalamnya terdapat identifier, konstanta, variabel, tipe data, dan dalam satu program ada operator unary (increment dan decrement), tunjukkan keterangan nya. mohon bantuannya


Soal :

tuliskan contoh satu program bahasa c++ yang didalamnya terdapat identifier, konstanta, variabel, tipe data, dan dalam satu program ada operator unary (increment dan decrement), tunjukkan keterangan nya. mohon bantuannya


Jawaban :

#include <iostream>

using namespace std;

int main( ){

   const int n=9;  //konstanta

   int x=5;  //variabel = x. tipe data = int

   x++;  //Increment

   cout << "Increment : " << (x*n) << endl;

   x--; //Decrement

   cout << "Decrement : " << (x*n) << endl;

   return 0;

}


Pembahasan :

Identifier terdiri atas konstanta dan variabel.

Konstanta bersifat tetap, tidak  berubah meskipun pada program terdapat operasi unary didalamnya. Pada program yang saya buat, konstanta nya adalah n=9 dengan tipe data bilangan bulat (int). Variabel digunakan untuk mewakili suatu nilai tertentu pada program. nilai tertentu tersebut tipe data. Pada program yang saya buat, variabel nya adalah x dengan tipe data bilangan bulat (int).

Tipe data terdiri dari bilangan bulat (short, unsigned short, int, unsigned int, long long, dan unsigned long long), bilangan riil (double dan float), karakter (char) dan boolean (true dan false).

Operator unary terdiri atas increment dan decrement. Pada program yang saya buat, operator incrementnya adalah x++ dan operator decrementnya adalah x--.

Output pada program tersebut ada pada gambar.

Pada output program tersebut, nilai increment nya menjadi 54 karena nilai x yang awal nya 5 menjadi 6 karena mendapat operator unary incrementnya, lalu nilai x setelah mendapat operator unary dikalikan dengan konstanta n yang bernilai 9. Sehingga output menjadi Increment : 54.

Karena nilai x sekarang 6, setelah mendapat operator unary decrement menjadi 5. Lalu nilai x setelah mendapat operator unary dikalikan dengan konstanta n yang bernilai 9. Sehingga output nya menjadi Decrement : 45
