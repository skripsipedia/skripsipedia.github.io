3. Buatlah program java looping untuk soal dibawah ini


Program dalam bahasa java

// AUTHOR : LordRimuruTempest

public class main {

   public static void main(String[] args) {

       int temp = 1;

       for (int i = 1; i <= 15; i++) {

           System.out.print(i + " ");

           if(i == ((temp + 1) * temp) / 2){

               System.out.println();

               temp += 1;

           }

       }

   }

}

Pembahasan

Untuk membuat output seperti pada soal, perlu diketahui dulu cara kerja dari program tersebut. Perhatikan bahwa bilangan terakhir pada tiap baris memiliki pola sebagai berikut

1 = 1

3 = 1 + 2

6 = 1 + 2 + 3

10 = 1 + 2 + 3 + 4

15 = 1 + 2 + 3 + 4 + 5

atau dengan kata lain, bilangan terakhir pada baris ke-n adalah

1 + 2 + 3 + ... + n = ((n + 1) x n) / 2

Sehingga alur program diatas dapat ditulis sebagai berikut :

1. Program dimulai

2. Buat variabel temp untuk menyimpan nomor baris

3. Beri nilai awal temp = 1

2. lakukan perulangan dari indeks i = 1 sampai i = 15

cetak i tanpa membuat baris barujika i = ((temp + 1) x temp) / 2 maka ganti baris, dan nomor baris bertambah 1 yaitu temp = temp + 1

5. Program selesai

Alur/algoritma program dapat ditulis dalam pseudocode sebagai berikut

Algoritma program

program start

var temp : integer

temp <-- 1

for i = 1 to i = 15 do

   print i  //no new line

   if(i = ((temp + 1) * temp) / 2) then

       //new line

       temp <-- temp + 1

   end if

end for

program end

Untuk menerapkan algoritma diatas ke dalam program java, pertama-tama buat class java(nama kelas bebas) beserta main method-nya. Deklarasikan variabel temp dengan tipe integer dan beri nilai awal temp = 1. Kemudian lakukan perulangan(looping) menggunakan for sesuai dengan arahan dari algoritma/alur program. Program beserta output-nya dapat diamati pada foto yang sudah dilampirkan.

6. Tuliskan program sederhana bahasa java


Program dalam bahasa java

// AUTHOR : LordRimuruTempest

import java.util.Scanner;

public class menghitungMean {

   public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       double data, rataRata, jumlahData = 0;

       System.out.print("Masukkan banyak data : ");

       int banyakData = input.nextInt();

       for (int i = 1; i <= banyakData; i++) {

           System.out.print("Masukkan data ke " + i + " : ");

           data = input.nextDouble();

           jumlahData += data;

       }

       rataRata = jumlahData / banyakData;

       System.out.println("Nilai rata-ratanya adalah : " + rataRata);

   }

}

Pembahasan

Bahasa java dapat digunakan untuk membuat program-program sederhana yang dapat membantu pekerjaan manusia. Salah satu contohnya adalah program menghitung rata-rata dari n data. Perhatikan bahwa rumus dari rata-rata(mean) adalah sebagai berikut :

[tex]\boxed{\bold{rata-rata = \frac{jumlah \hspace{1 mm} data}{banyak \hspace{1 mm} data} }}[/tex]

Maka algoritma program dalam bentuk pseudocode dapat ditulis sebagai berikut:

Algoritma menghitungMean

program start

var : data, rataRata, jumlahData : double

var : banyakData : integer

jumlahData <-- 0

read banyakData

for i = 1 to i = banyakData do

     read data

     jumlahData <-- jumlahData + data

end for

rataRata <-- jumlahData / banyakData

print rataRata

program end

Setelah alur program dibuat, maka langkah selanjutnya adalah menerapkan algoritma ke dalam bahasa pemrograman java. Pertama-tama import library java.util.Scanner untuk menggunakan objek scanner(untuk meminta input user) dan buat kelas bernama menghitungMean beserta main method-nya. Buat satu objek scanner java dengan nama input untuk memasukkan input dari user. Deklarasikan variabel data, jumlahData, dan rataRata bertipe double serta banyakData bertipe integer. Isi jumlahData dengan nilai awal 0 dan minta user memasukkan banyak data. Lakukan perulangan for untuk melakukan input data serta menghitung jumlah datanya. Terakhir tinggal menghitung rata-ratanya sesuai rumus yang berlaku kemudian nilai rata-ratanya dicetak. Program beserta output-nya dapat diamati pada lampiran foto.

12. Membuat Program Java Persegi Bintang​


public class Segitiga {

 public static void main(String[] args) {

   // Dimensi segitiga

   int tinggi = 5;

   // Looping untuk menampilkan baris

   for (int i = 0; i < tinggi; i++) {

     // Looping untuk menampilkan kolom

     for (int j = 0; j < tinggi * 2 - 1; j++) {

       // Jika kolom kurang dari atau sama dengan baris

       if (j >= tinggi - i - 1 && j <= tinggi + i - 1) {

         // Tampilkan karakter '*'

         System.out.print("*");

       } else {

         // Jika tidak, tampilkan karakter spasi

         System.out.print(" ");

       }

     }

     // Pindah ke baris selanjutnya

     System.out.println();

   }

 }

}

1. Harus disimpan berekstensi *.java

2. Nama file harus sama dengan nama class public, misalnya nama filenya myHello.java, maka nama class public juga harus myHello

Contoh:

public class myHello {

//harus sama dengan nama file (myHello.java)

}

3. Comment sebaiknya sebaiknya ditulis untuk menjelaskan sebuah class atau method. Comment ditulis untuk tujuan dokumentasi.

Contoh:

public static void main(String[]args){

//untuk menampilkan kata myfirs java

System.out.println(“Hello this is my first java”);

}

4. Java statement adalah suatu baris yang diakhiri dengan titik koma.

Contoh:

System.out.println(“Hello this is my first java”);

public static void main(String[]args){
