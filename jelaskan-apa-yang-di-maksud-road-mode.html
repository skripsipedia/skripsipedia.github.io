14. jelaskan perbedaan object mode dan edit mode


Kelas           : XI (2 SMA)

Pelajaran     : TI

Kategori      : Dasar Dasar Blender

Kata Kunci  : Grafis 3D, Object Mode, Edit Mode

 

 

Object Mode dan Edit Mode adalah dua jenis mode yang bisa kita jumpai saat hendak menambah objek pada program Blender. Kedua mode ini mempunya fungsi yang berbeda.

 

❖  Edit Mode dipakai untuk melakukan kegiatan pengeditan pada suatu objek dengan cara memilih titik titik tertentu.

❖  Object Mode dipakai untuk melakukan kegiatan pengeditan pada suatu objek secara menyeluruh.



Catatan:

 

Blender merupakan program grafis 3D yang dipakai dalam membuat suatu animasi, model cetak 3D, efek visual, aplikasi 3D dan masih banyak lagi lainnya. Program Blender ini menawarkan beberapa fitur seperti permodelan 3D, penyunting gambar bitmap, simulasi cairan juga asap, penteksturan, simulasi partikel, animasi, pemahat digital, rendering dan penyunting video.

19. apa yang dimaksud kota mode​


Jawaban:

Kota mode adalah sebuah kota yang menjadi pusat utama industri mode, kegiatannya termasuk desain mode, produksi, ritel produk-produk mode, ajang mode (seperti pekan mode dan penghargaan mode), serta pameran dan aktivitas perdagangan yang terkait dengan mode yang menghasilkan pemasukan perekonomian yang signifikan bagi kota tersebut. Kota mode umumnya memiliki subbudaya kuat yang mampu memberikan inspirasi dan menjadi tren, tidak hanya bagi kalangan profesional mode, namun juga bagi warga kota yang bersangkutan, dan mengubah gaya dan budaya jalanan mereka menjadi karakteristik penting. Kota mode biasanya juga memiliki bisnis mode yang maju, hiburan, budaya, dan aktivitas rekreasi yang signifikan dan diakui secara internasional atas identitasnya yang unik dan kuat.[1]

Saat ini, ada empat kota yang dianggap sebagai pusat mode dunia, yang dikenal dengan julukan "empat besar", yaitu London, Milan, New York City dan Paris.[2][3][4][5] Peringkat mengenai kota mode utama di dunia dirilis oleh Global Language Monitor setiap tahunnya.[6]

33. 6. Jelaskan yang dimaksud dengan incognito mode dan mafaat dari penggunaan fitur ini!7. Jelaskan kelebihan dan kekurangan menggunakan incognito mode ketika menjelajahi Internet!8. Jelaskan yang dimaksud dengan bookmarks!​


6. Mode incognito singkatnya adalah mode penyamaran. Mode ini dirancang untuk menghapus data lokal apapun pada sesi pencarian situs. Itu berarti, cookie apapun yang coba diunggah situs ke perangkat Anda akan diblokir atau dihapus. Tidak ada satupun riwayat pencarian yang tertinggal, file sementara, tracker, dan alat toolbar pihak ketiga dinonaktifkan.

7. Kelebihan : Dengan menggunakan Incognito Mode, riwayat penelusuran kita tidak akan tersimpan, sehingga dengan menggunakan mode ini, kita akan mendapatkan privasi, kenyamanan, dan keamanan yang lebih baik saat berselancar di internet.  

Kelemahan :menggunakan fitur incognito mode ini membuat kecepatan browsing menjadi sedikit lebih lambat jika di bandingkan saat tidak menggunakannya. Hal ini terjadi karena Incognito Mode tidak akan menyimpan cache, cookies dan history dari website yang kita akses. Ia akan memulai sesi baru ketika kita menutup window web browser.

8. Bookmarks adalah sebuah fasilitas pada browser (mozilla,internet explorer,opera,chrome) untuk menyimpan nama atau alamat website sehingga kita tidak perlu menghapal nama-nama website yang begitu banyak.

36. apa yang dimaksud dengan mode AP Bridge dan mode Client/Station ?


1.-Dalam mode ini, anda dapat menggunakan router broadband ini sebagai bridge jaringan nirkabel dan mengijinkan semua komputer yang terhubung ke port LAN kedua Access Point router nirkabel tersebut untuk berkomunikasi dengan satu sama lain. Mode ini hanya mendukung satu titik access point nirkabel, artinya komunikasi tersebut hanya terjadi antara dua unit router nirkabel sebagai bridge (jembatan).poin to poin
-Dalam mode ini, Anda dapat menggunakan router broadband nirkabel ini sebagai jembatan jaringan dan menghubungkan akses semua komputer ke port LAN di setiap router nirkabel  untuk dapat berkomunikasi dengan satu sama lain. Mode ini dapat menghubungkan hingga empat Access Point nirkabel.yg ini multi poin.  
2.Mode Station yaitu konfigurasi yang dilakukan pada router sekunder (kedua) yang akan dijadikan sebagai wireless client, caranya dengan melakukan site survey dan memilih SSID (nama wifi) wireless access point dari router utama. Selain itu mode ini juga berfungsi sebagai wireless access point untuk melayani wireless client lainnya. Jika anda memiliki hak akses ke wireless  access point lain terdekat dan anda ingin menggunakan titik akses untuk mengakses Internet, anda dapat menggunakan jenis koneksi ini.
