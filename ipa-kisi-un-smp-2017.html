8. Persiapan UN 2017 Diutamakan siswa - siswi SMP Gunakanlah cara / langkah yang baik


Mapel : Matematika
Kelas : IX SMP
Bab : Barisan dan Deret

Pembahasan :
Barisan Geometri
U1 + U3 = x
[tex]a + ar^2 = x[/tex]

U2 + U4 = y
[tex]ar + ar^3 = y[/tex]

[tex] \frac{x}{y} = \frac{(a + ar^2)}{(ar + ar^3)} [/tex]
[tex] \frac{x}{y} = \frac{a(1 + r^2)}{ar(1 + r^2)} [/tex]
[tex] \frac{x}{y} = \frac{1}{r} [/tex]
[tex] \frac{y}{x} = r[/tex]

U1 + U3 = x
[tex]a(1 + r^2) = x[/tex]
[tex]a = \frac{x}{(1 + r^2)} [/tex]
Substitusi r = y/x
[tex]a = \frac{x}{(1 + \frac{y^2}{x^2}) } [/tex]
[tex]a = \frac{x}{( \frac{x^2 + y^2}{x^2}) } [/tex]
[tex]a = \frac{x^3}{(x^2 + y^2)} [/tex]

Jadi... [tex]U1 = \frac{x^3}{(x^2 + y^2)} [/tex]


U1 + U3 = x
a + ar^2 = x
a(1+r^2) = x...............(1)

U2 + U4 = y
ar + ar^3 = y
ar(1+r^2) = y.............(2)

bandingkan persamaan (1) dan (2)
x/y = a(1+r^2) / ar(1+r^2)
x/y =a/ar
x/y = 1/r
r = y/x.........................(3)

pilih persamaan (1) atau (2)
mis = persamaan (1)
U1 + U3 = x
a(1+r^2) = x
a = x / (1+r^2)

subtitusikan perasamaan (3) pada persamaan (1)
a = x / (1+r^2)
=x / (1+(y/x)^2)
=x / (( x^2 + y^2) / x^2))
= x^3 / (x^2 + y^2)

dengan demikian nilai U1 adalah
U1 = a
= x^3 / (x^2 + y^2)

11. soal UN matematika IPA , 11 april 2017


Aplikasi Gerak Parabola
Fungsi Kuadrat
UN SMA IPA 2017

Step-1

y = - 2t² + 120t + 15
15 = - 2t² + 120t + 15
- 2t² + 120t = 0
t² - 60t = 0
t(t - 60) = 0
t = 0 ⇒ kondisi awal
t = 60 ⇒ seharusnya tepat sasaran

Step-2

x = 30t
dengan t = 60 sekon
x = 30(60)
x = 1800 m
 
Final Step

Perbedaan jarak ⇒ 1900 - 1800 = 100

Kesimpulan

⇒ Peluru belum tepat mencapai sasaran
⇒ Jatuhnya peluru berjarak 100 m di depan D

38. Persiapan UN 2017 Diutamakan siswa - siswi SMP Gunakanlah cara / langkah yang baik


garis P (3,1) dan (9,3)
persamaannya :
(y-1)/(3-1) = (x-3)/(9-3)
(y-1)/2 = (x-3)/6
6(y-1) = 2(x-3)
6y - 6 = 2x -6
6y = 2x
y = ⅓x

garis S (6,0) dan (0,2)
persamaannya :
y/2 = (x-6)/(0-6)
y/2 = (x-6)/-6
-6y = 2x-12
y = -⅓x +2

⅓x = -⅓x +2
⅔x = 2
x = 2 . (3/2) = 3
y = ⅓.3 = 1

kedua garis berpotongan di (3,1)garis P
(y-y1)/(y2-y1) = (x-x1)/(x2-x1)
(y-1)/(3-1) = (x-3)/(9-3)
(y-1)/2 = (x-3)/6
(y-1) = (x-3)/3
3y-3 = x - 3
3y=x................(1)

garis S
(y-y1)/(y2-y1) = (x-x1)/(x2-x1)
(y-0)/(2-0) = (x-6)/(0-6)
-3y=x-6
3y= 6-x...............(2)

subtitusi persamaan (2) ke persamaan (1)
3y=6-x
x=6-x
2x=6
x=3

subtitusikan x=3 kepersamaan (1)
3y=x
3y=3
y=1

maka titik potong garis P dan S adalah (3,1)
