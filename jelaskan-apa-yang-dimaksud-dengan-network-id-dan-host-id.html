1. Jelaskan apa yang dimaksud dengan Network id dan host id dan berikan contohnya


Pada dasarnya Network ID sendiri dapat kita artikan sebagai suatu Identitas dari suatu kelompok IP Address atau subnet, yang mana Network ID ialah suatu bagian dari IP address yang menunjukan letak network atau jaringan dari suatu IP Address. Sedangkan untuk Host ID sendiri ialah suatu nomor perangkat keras fisik Ethernet Network Interface Card atau yang sering disingkat dengan sebutan NIC.

Sebagai salah satu contohnya ialah:

Kelas A yaitu 10.11.12.1 maka untuk contoh ini network ID nya ialah 10. sedangkan untuk Host ID ialah 11.12.1.Kelas B yaitu 188.10.11.1 maka untuk contoh ini network ID nya ialah 188.10. sedangkan untuk Host ID ialah 11.1.Kelas C yaitu 192.168.1.1 maka untuk contoh ini network ID nya ialah 192.168.1 sedangkan untuk Host ID ialah 1.

Pembahasan

Pada umumnya Net ID sendiri dapat berperan penting dalam mengidentifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network, yang mana pada konteks ini dapat kita simpul kan bahwa seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama.

32. Network id, broadcast id, range ip, jumlah host dari ip 192.168.45.1/24


IP address : 192.168.45.1/24 atau

= 192.168.45.1 255.255.255.0

Cara menghitung :

Ubah IP Address dan Netmask kedalam bentuk biner, 192.168.45.1 IP Address tersebut berada di kelas C,  ambil octet ketiga dan ke empat untuk dihitung, maka :

IP address : x.x.0010 1101. 0000 0001 = 45.1

Netmask : x.x.1111 1111. 0000 0000 = 255.0

Jawab:

1. Network ID

Network ID = IP Address AND Netmask

IP address : x.x.0010 1101. 0000 0001

Netmask    : x.x.1111 1111. 0000 0000

——————————————————

AND           : x.x.0010 1101. 0000 0000 = 45.0

Sehingga Network ID nya adalah 192.168.45.0

2. Broadcast ID

Broadcast ID = IP Address OR ~Netmask

IP address : x.x.0010 1101. 0000 0001

~Netmask  : x.x.0000 0000 . 1111 1111

———————————————————

OR : x.x.0010 1101. 1111 1111 = 45.255

Sehingga Broadcast Id nya adalah 192.168.45.255

3. Range IP

Range IP = Network ID hingga(-) Broadcast ID

Sehingga Range IP nya adalah 192.168.45.0 – 192.168.45.255

4. Jumlah Host

Jumlah Host = [tex]2^{y} -2[/tex]

dimana y adalah jumlah biner 0 pada octet 3 dan 4 netmask

/24 bila di-binerkan menjadi:

1111 1111. 1111 1111. 1111 1111. 0000 0000

255         255        255            0

Sehingga Jumlah Host

= [tex]2^{8} - 2[/tex]

= [tex]256 - 2[/tex]

= [tex]254[/tex]


33. Pengertian network id, host id, dan broadcast id


Pengertian✍️

Network id

Network ID adalah bagian dari IP address yang menunjukkan di jaringan mana komputer tersebut berada.

Host id

Host adalah identitas dari host (interface router, workstation, dan device) yang terhubung ke jaringan.

Broadcast id

broadcast adalah alamat jaringan yang digunakan untuk mengirim ke semua perangkat yang terhubung ke jaringan komunikasi akses ganda.
