Diketahui :

[tex]\Huge{\sf{\dfrac{2\sqrt{8}}{4\sqrt{2} - 2\sqrt{6}}}}[/tex]

[tex] \: [/tex]

Ditanya :

Bentuk rasional ....?

[tex] \: [/tex]

Jawaban :

[tex]{\sf{\frac{2\sqrt{8}}{4\sqrt{2} - 2\sqrt{6}}}}[/tex]

[tex]{\sf{ = \frac{\bcancel2\sqrt{8}}{\bcancel2(2\sqrt{2} - \sqrt{6})}}}[/tex]          ....(Coret 2)

[tex]{\sf{ = \frac{\sqrt{8}}{2\sqrt{2} - \sqrt{6}}}}[/tex]

[tex]{\sf{ = \frac{\sqrt{8}}{2\sqrt{2} - \sqrt{6}} \times \frac{2\sqrt{2} + \sqrt{6}}{2\sqrt{2} + \sqrt{6}}}}[/tex]

[tex]{\sf{ = \frac{\sqrt{8}(2\sqrt{2} + \sqrt{6})}{(2\sqrt{2} - 6)^2(2\sqrt{2} + 6)^2}}}[/tex]

[tex]{\sf{ = \frac{\sqrt{4 \times 2}(2\sqrt{2} + \sqrt{6})}{4 \times 2 - 6}}}[/tex]

[tex]{\sf{ = \frac{\sqrt{4} \times \sqrt{2}(2\sqrt{2} + \sqrt{6})}{8 - 6}}}[/tex]

[tex]{\sf{ = \frac{\bcancel2\sqrt{2}(2\sqrt{2} + \sqrt{6})}{ \: \bcancel2}}}[/tex]          ....(Coret 2)

[tex]{\sf{ = \sqrt{2}(2\sqrt{2} + \sqrt{6})}}[/tex]

[tex]{\sf{ \approx 4 + 2\sqrt{3}}}[/tex]

[tex] \: [/tex]

ᴋᴇꜱɪᴍᴘᴜʟᴀɴ

Jadi, bentuk rasional dari [tex]{\sf{\frac{2\sqrt{8}}{4\sqrt{2} - 2\sqrt{6}}}}[/tex] adalah [tex]{\sf{4 + 2\sqrt{3}}}[/tex]


28. 10. PK yang akar-akarnya kurang 3 dari akar-akar pk x² + 5x - 7 = 0 adalah....Tolong bantu jawab yah kakak", soalnya besok mau dikumpulkan​


Jawab:

﻿x² + 11x + 17 = 0

Penjelasan dengan langkah-langkah:

﻿ax² + bx + c = 0

﻿x² + 5x - 7 = 0

x﻿₁ + x₂ = -b/a = -5/1 = -5

﻿x₁x₂ = c/a = -7/1 = -7

﻿α = x₁ - 3

﻿β = x₂ - 3

﻿x² - (α + β)x + αβ = 0

﻿x² - (x₁ - 3 + x₂ - 3)x + (x₁ - 3)(x₂ - 3) = 0

﻿x² - (x₁ + x₂ - 6)x + x₁x₂ - 3(x₁ + x₂) + 9 = 0

﻿x² - (-5 - 6)x - 7 - 3(-5) + 9 = 0

﻿x² + 11x + 17 = 0

﻿Cara cepat

﻿Persamaan kuadrat ax² + bx + c = 0 akar-akarnya x₁ atau x₂. Persamaan kuadrat yang akar-akarnya x₁ - m atau x₂ - m adalah a(x + m)² + b(x + m) + c = 0

﻿1(x + 3)² + 5(x + 3) - 7 = 0

﻿x² + 6x + 9 + 5x + 15 - 7 = 0

﻿x² + 11x + 17 = 0

Pendahuluan:

Persamaan kuadrat adalah persamaan yang pangkat tertinggi dari variabelnya adalah dua

Persamaan kuadrat memiliki beberapa fungsi, diantaranya:

mengetahui kecepatan rata-rata sebuah kendaraan yang melaju dengan kecepatan berbeda-bedamengetahui jatuhnya sebuah bola ke tanah jika dilambungkan ke atas

┈

Bentuk umum persamaan kuadrat

[tex] \begin{array} { |c|} \hline \rm ax^2 + bx + c = 0\\ \hline\end{array} [/tex]

keterangan:

[tex] \rm a, b \: dan \: c [/tex] bilangan real[tex] \rm a [/tex] koefisien [tex] \rm x^2 [/tex][tex] \rm b [/tex] koefisien [tex] \rm x [/tex][tex] \rm c [/tex] konstanta[tex] \rm a ≠ 0 [/tex]

-

Ciri-ciri persamaan kuadrat

memiliki satu variabel yang berpangkat duatidak memiliki variabel berpangkat lebih dari dua

┈

Jenis-jenis persamaan kuadratpersamaan kuadrat biasa. contoh: [tex] \rm x^2 + 5x - 3 = 0 [/tex]persamaan kuadrat tidak lengkap. contoh: [tex] \rm 2x^2 + 5x = 0 [/tex]persamaan kuadrat murni. contoh: [tex] \rm x^2 - 6 = 0 [/tex]

┈

Penyelesaian persamaan kuadrat

Metode faktorisasi

persamaan kuadrat ax² + bx + c = 0 dapat difaktorkan dalam bentuk

[tex] \boxed{ \rm \footnotesize ax^2 + bx + c = (ax + p) \left( x + \frac{q}{a} \right) }[/tex]

dengan

[tex] \rm b = p + q [/tex][tex] \rm c = \frac{pq}{a}[/tex]

-

Metode melengkapkan kuadrat sempurna

Tahap untuk melengkapkan kuadrat sempurna, sebagai berikut:

Pisahkan konstanta atau pindahkan konstanta ke ruas kananJika a ≠ 1, bagi kedua ruas dengan aTambahkan pada ruas kuadrat dari [tex] \frac{1}{2}[/tex] kali koefisien xNyatakan dalam bentuk kuadrat sempurna pada ruas kiriTentukan penyelesaian persamaan kuadrat sempurna di atas dengan menarik akar

-

Metode rumus abc

[tex] \boxed{\rm x = \frac{-b ± √b² - 4ac}{2a}} [/tex]

Operasi jumlah dan hasil kali akar-akar persamaan kuadrat

jika [tex]\rm x_1 [/tex] dan [tex]\rm x_2 [/tex] merupakan akar persamaan kuadrat dari ax² + bx + c = 0, dengan [tex] \rm \rm a, b, c ∈ R [/tex] dan [tex] \rm a ≠ 0[/tex], maka

[tex]\begin{array} { |c|c|} \hline \tiny\diamondsuit&\scriptsize \rm x_1 + x_2 = - \frac{b}{a} \\ \tiny\diamondsuit&\scriptsize \rm x_1 - x_2 = \frac{ \sqrt{D} }{a} \\ \tiny\diamondsuit&\scriptsize \rm x_1 x_2 = \frac{c}{a}\\ \tiny\diamondsuit&\scriptsize \rm \frac{1}{x_1} + \frac{1}{x_2} = - \frac{b}{c}\\ \tiny\diamondsuit&\scriptsize \rm \frac{1}{x_1} - \frac{1}{x_2} = \frac{ \sqrt{D} }{c}\\ \tiny\diamondsuit&\scriptsize \rm x_ 1{}^{2} + x_2 {}^{2} = {(x_ 1 + x_ 2)}^{2} - 2x_ 1x_ 2\\ \tiny\diamondsuit&\scriptsize \rm x_ 1{}^{2} - x_2 {}^{2} = (x_ 1 + x_ 2)(x_ 1 - x_ 2)\\\tiny\diamondsuit &\scriptsize \rm {(x_ 1 - x_ 2)}^{2} =x_ 1{}^{2} + x_2 {}^{2} - 2x_ 1 x_ 2 \\\tiny\diamondsuit&\scriptsize \rm x_ 1{}^{3} + x_2 {}^{3} = {(x_ 1 + x_ 2)}^{3} - (3x_ 1x_ 2(x_ 1 + x_ 2))\\\tiny\diamondsuit&\scriptsize \rm x_ 1{}^{4} + x_2 {}^{4} = ( { x_ 1 + x_ 2) }^{4} - 2 x_ 1 x_ 2(2( x_ 1 {}^{2} + { x_ 2}^{2}) + 3 x_ 1 x_ 2)\\ \hline\end{array} [/tex]

Menentukan jumlah dan hasil kali akar-akarnya

Dijawab:

[tex] \rm 2x^2-6x+18 = 0 [/tex]

[tex] \footnotesize \rm a = 2 [/tex][tex] \footnotesize \rm b = -6 [/tex][tex] \footnotesize \rm c = 18[/tex]

-

[tex] \scriptsize{ \rm Menentukan\: hasil \: jumlah \: akar-akar }[/tex]

[tex] \rm\begin{aligned} \footnotesize \rm x_{_{1}} + x_{_{2}} &= \footnotesize\rm - \frac{b}{a}\\ & =\footnotesize \frac{6}{2} \\ & =\footnotesize \bold{3} \end{aligned}[/tex]
