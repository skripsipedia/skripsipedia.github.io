Penjelasan dengan langkah-langkah:

a. Persamaan garis y = 2x + 5 

   jika x = 0 →  y = 2x + 5

                        y = 2 (0) + 5

                        y = 0 + 5

                        y = 5

           titik (0 , 5)

  jika y = 0 → y = 2x + 5

                      0 = 2x + 5

                    -2x = 5

                       x = - 5/2

             titik (-5/2 , 0)

b. Persamaan garis y = 4x - 1

   jika x = 0 →  y = 4x - 1

                        y = 4 (0) - 1

                        y = 0 - 1

                        y = -1

           titik (0 , -1)

  jika y = 0 → y = 4x - 1

                      0 = 4x - 1

                       1 = 4x

                       x = 1/4

             titik (1/4 , 0)

11. suatu fungsi linear dangan f(0)= -6 dan f(-3)=-5 tentukan rumus fungsinya. tolong jawab yah soalnya mau dikumpul besok ​


Suatu fungsi linear dengan [tex] \rm f(0) =-6[/tex] dan [tex] \rm f(-3) = -5[/tex]. Maka rumus fungsinya adalah [tex] \rm f(x) = -\frac{1}{3}x -6[/tex].

Pendahuluan

[tex] \sf \blacktriangleright Pengertian~Fungsi [/tex]

Fungsi adalah suatu relasi yang memetakan setiap anggota dari daerah asal (domain) ke tepat satu anggota di daerah kawan (kodomain).

Jika f : x ---> ax + b dengan x anggota domain f , maka rumus fungsi adalah

[tex] \boxed{\rm f(x) = ax + b}[/tex]

[tex]\\[/tex]

Pembahasan

Diketahui :

[tex] \rm f(0) = -6[/tex][tex] \rm f(-3) =- 5[/tex]

Ditanya :

Rumus fungsi?

Jawab :

Modelkan persamannya :

[tex] \rm f(0) = -6 \rightarrow 0a + b = -6\rightarrow b = -6[/tex] ...(1)

[tex] \rm f(-3) = 5 \rightarrow -3a+b = -5[/tex]...(2)

Dari persamaan (1) sudah diketahui nilai b. Maka langkah selanjutnya mencari nilai a :

[tex] \rm -3a + (-6) = -5[/tex]

[tex] \rm -3a -6 = -5[/tex]

[tex]\rm -3a = -5+6[/tex]

[tex] \rm -3a = 1[/tex]

[tex] \displaystyle \rm a = -\frac{1}{3}[/tex]

Bentuk rumus fungsi :

[tex] \rm f(x) = ax+b[/tex]

[tex] \rm f(x) = -\frac{1}{3}x +(-6)[/tex]

[tex] \rm f(x) = -\frac{1}{3}x -6[/tex]

Kesimpulan

Jadi, diperoleh rumus fungsinya adalah [tex] \rm f(x) = -\frac{1}{3}x -6[/tex].

15. contoh soal kombinasi linearjawab please besok kumpul nih



Diketahui a = (1, 2), b = (-2, -3), dan c = (1, 3). Apakah c merupakan kombinasi linear dari a dan b?

Jawab:

Misalkan c merupakan kombinasi linear dari a dan b maka dapat ditentukan dengan c = k1a + k2b

(1, 3) = k1(1, 2) + k2(-2, -3)

(1, 3) = (1k1, 2k1) + (-2k2, -3k2)

Maka dapat dinyatakan 1 = k1 – 2k2 dan 3 = 2k1 – 3k2 Sehingga diperoleh pengenyelesaian k1 = 3 dan k2 = 1

Jadi c merupakan kombinasi linear dari a dan b, dan dinyatakan dengan c = 3a + b

1.
Sekarang umur seorang adik 5 tahun kurangnya dari umur kakak. Lima tahun kemudian jumlah umur kakak dan adik menjadi 35 tahun. Tentukanlah masing-masing umurnya.

Penyelesaian :

Misalkan : Umur kakak = x tahun
                    Umur adik = (x - 5) tahun
5 tahun kemudian
umur kakak = x + 5 tahun
   umur adik = (x - 5) + 5 = x tahun
Jumlah umur mereka 5 tahun lagi adalah 35 tahun,
maka kalimat matematikanya adalah:
x + 5 + x = 35,                              kita lanjutkan penyelesaiannya     
    2x + 5 = 35          
          2x = 30
            x = 30/2           
            x = 15
Jadi, umur kakak sekarang adalah 15 tahun dan adik adalah 15 – 5 = 10 tahun.
